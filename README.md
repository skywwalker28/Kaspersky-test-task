# File Indexer
Библиотека для индексации текстовых файлов по словам с поддержкой поиска, многопоточного доступа и отслеживания изменения файловой системы.

## Возможности

- добавление файла в индекс
- добавление директории в индекс
- поиск файлов по слову
- автоматическое обновление индекса при изменении файлов
- удаление файла из индекса
- расширяемый механизм токенизации через интерфейс `Tokenizer`

## Требования
- Java 21+
- Maven 3.9+
 

## Сборка
```
mvn clean install
```

## Запуск CLI
```
mvn exec:java -Dexec.mainClass="com.kaspersky.fileindexer.cli.Main"
```
или запуском Main из IDE.

## Команды CLI
- add-file <path>
- add-dir <path>
- search <word>
- exit

## Запуск тестов
```
mvn test
```

## Поддерживаемые файлы
Сейчас индексируется только ```.txt``` файлы

## Архитектура
- ```FileIndexService``` - API библиотеки
- ```InMemoryFileIndexService``` - in-memory реализация индекса
- ```Tokenizer``` - интерфейс разбиения текста на слово
- ```SimpleTokenizer``` - простая реализация токенизации
- ```DirectoryWatcher``` - отслеживание изменений в директориях
- ```Main``` - консольное приложение
